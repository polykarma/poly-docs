Properties of Relations

In particular there are a few properties that we like to study when we talk about relations. These properties indicate the nature of the relation and can be used to classify them. Such as the Equivalence relation, and the Partial order relation.

Equivalence relation​

An equivalence relation is a relation that is reflexive, symmetric and transitive.

Reflexive​

A relation RRR on a set SSS is reflexive if for all a∈Sa \in Sa∈S, (a,a)∈R(a, a) \in R(a,a)∈R.

This can be seen in a graph as a self-loop on each node.

For example, the relation R={(a,a),(b,b),(c,c)}R = \{(a, a), (b, b), (c, c)\}R={(a,a),(b,b),(c,c)} is reflexive.

In a matrix representation, we can see that the diagonal is filled with 1s.

[100010001]\begin{bmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{bmatrix}​100​010​001​​

Symmetric​

A relation RRR on a set SSS is symmetric if for all a,b∈Sa, b \in Sa,b∈S, if (a,b)∈R(a, b) \in R(a,b)∈R then (b,a)∈R(b, a) \in R(b,a)∈R.

This can be seen in a graph as a bidirectional edge between two nodes.

For example, the relation R={(a,b),(b,a),(c,c)}R = \{(a, b), (b, a), (c, c)\}R={(a,b),(b,a),(c,c)} is symmetric.

In a matrix representation, we can see that the matrix is symmetric using the diagonal as the axis of symmetry.

[010100001]\begin{bmatrix} 0 & 1 & 0 \\ 1 & 0 & 0 \\ 0 & 0 & 1 \end{bmatrix}​010​100​001​​

Transitive​

A relation RRR on a set SSS is transitive if for all a,b,c∈Sa, b, c \in Sa,b,c∈S, if (a,b)∈R(a, b) \in R(a,b)∈R and (b,c)∈R(b, c) \in R(b,c)∈R then (a,c)∈R(a, c) \in R(a,c)∈R.

This can be seen in a graph as a direct edge between any two nodes that are connected by a path.

For example, the relation R={(a,b),(b,c),(a,c)}R = \{(a, b), (b, c), (a, c)\}R={(a,b),(b,c),(a,c)} is transitive.

In a matrix representation, we can see that if there is a 1 in the row of a node, then there is a 1 in the column of the node that is connected to it.

[011001000]\begin{bmatrix} 0 & 1 & 1 \\ 0 & 0 & 1 \\ 0 & 0 & 0 \end{bmatrix}​000​100​110​​

Congruence mod m relation​

A congruence mod m relation is an equivalence relation on the set of integers.

For example:

R={(a,b)∈Z×Z∣a≡b(modm)}R = \{(a, b) \in \mathbb{Z} \times \mathbb{Z} \mid a \equiv b \pmod{m}\}R={(a,b)∈Z×Z∣a≡b(modm)}

where,

a≡b(modm)  ⟷  a(modm)=b(modm)a \equiv b \pmod{m} \ \ \longleftrightarrow \ \ a \pmod{m} = b \pmod{m}a≡b(modm)  ⟷  a(modm)=b(modm)

Why?

Because the congruence mod m relation is reflexive, symmetric and transitive.

Reflexive demonstration:

if

a≡a(modm)a \equiv a \pmod{m} a≡a(modm)

then,

a(modm)=a(modm)a \pmod{m} = a \pmod{m}a(modm)=a(modm)

so it's reflexive.

Symmetric demonstration:

if

a≡b(modm)a \equiv b \pmod{m}a≡b(modm)

then,

b≡a(modm)b \equiv a \pmod{m}b≡a(modm)

so it's symmetric.

Transitive demonstration:

if

a≡b(modm)a \equiv b \pmod{m}a≡b(modm)

and

b≡c(modm)b \equiv c \pmod{m}b≡c(modm)

then,

a≡c(modm)a \equiv c \pmod{m}a≡c(modm)

so it's transitive.

Partial order relation​

A partial order relation is a relation that is reflexive, antisymmetric and transitive.

Antisymmetric​

A relation RRR on a set SSS is antisymmetric if for all a,b∈Sa, b \in Sa,b∈S, if (a,b)∈R(a, b) \in R(a,b)∈R and (b,a)∈R(b, a) \in R(b,a)∈R then a=ba = ba=b.

This means that in a graph, there are no bidirectional edges.

This also means that in a matrix, there is at least one entry wherein the symmetric entry is not equal.

[000100001]\begin{bmatrix} 0 & 0 & 0 \\ 1 & 0 & 0 \\ 0 & 0 & 1 \end{bmatrix}​010​000​001​​

There are two reasonings we can use here:

  • An edge points to and from the same node: self-loop and entry on the diagonal.
  • Any edge exists: There is no bidirectional edge and the corresponding symmetric entry is the inverse.
